Five-Tool Confluence AIO v3.5-SHORT

Read the chart in five seconds, then confirm. This card decodes the go/no-go, the HUD states, and the risk rules — every number verified against the strategy source.

The 5-second go / no-go — all three, or stand aside

01 · REGIME
Are you with the tide?
Bull arms longs · Bear arms shorts · Neutral = no trade.
02 · SCORE ≥ 55
Is the confluence there?
The Score L / Sv2 cell. Floor is 55 for both sides. Below it, the stack isn't there — pass.
03 · HALT
Is risk clear?
The status cell must read Ready. Anything Paused = the system is standing down. So do you.

Regime gate (the hard rule)

BullLongs armed. Shorts impossible.
BearShorts armed only if Allow shorts is ON.
NeutralNo entries either side. Wait.

In this build Allow shorts ships ON — but the short book is an unvalidated experiment (see footer).

AVWAP arrow

↑Price above the anchored VWAP — buyers in control (bullish).
↓Price below it — sellers in control. Shorts require ↓.

The gold line re-anchors on every regime flip — read it as "since this leg began," not absolute.

The ! flag = low sample

Shorts / Longs rowFewer than 100 trades — stats are an anecdote.

Honesty is the product.  This tool describes the present, it does not predict. Every figure is a % of equity or an R-multiple — never dollars. The long side is the tested workhorse; the short side is unvalidated and net-negative in testing — treat it as a research project regardless of the default toggle. A candidate is not a setup; below ~100 trades, you know nothing.
