# Archived Claude trading-operation skills

These skills were written for Hermes's retired owner-specific Python/FastAPI
trading cockpit. They are preserved as historical source material and no longer
auto-load from `.claude/skills/`. Each entry is intentionally named
`LEGACY_SKILL.md`, not `SKILL.md`, so a direct folder scan cannot mistake it
for active automation.

Several contracts intentionally conflict with the active Hermes Journal mobile
product—for example, the legacy prohibition on currency P&L and its systemd,
regime, and proposal-journal assumptions. Do not apply or update these skills
for mobile work.

The active contract is defined by the repository root `AGENTS.md` and
`docs/mobile/PRODUCT_BLUEPRINT.md`.
