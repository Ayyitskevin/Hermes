# Claude repository guidance

Hermes currently has no auto-loaded repository skills.

The former trading-operation skill library described the retired personal
desktop cockpit and conflicted with the active mobile product. It is preserved
for historical reference in `docs/legacy/claude-trading-operation-skills/` but
must not govern new work.

All agents should begin with the repository root `AGENTS.md` and the active
documents under `docs/mobile/`.
